# Comparison of Supervised Machine Learning Algorithms for Fake News Detection

Machine learning project for detecting fake news articles using Natural Language Processing (NLP) and multiple classification algorithms. This project was created by Svetlana Grabar, Jared Graham, Michael Boren, and Amber Giza as part of the Master of Science in Artificial Intelligence program at City University of Seattle.

## Models Used

* Naive Bayes
* Decision Tree
* Penalized Logistic Regression (GLMNET)

## Running the Project

Download the Kaggle Fake and Real News dataset (`Fake.csv` and `True.csv`), place both files in the project directory, install the required R packages, and run the notebook.
